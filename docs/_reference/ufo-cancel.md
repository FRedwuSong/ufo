## Usage

    ufo cancel SERVICE

## Description

Cancel creation or update of the ECS service.


## Options

```
[--sure], [--no-sure]        # By pass are you sure prompt.
[--verbose], [--no-verbose]  
[--mute], [--no-mute]        
[--noop], [--no-noop]        
[--cluster=CLUSTER]          # Cluster.  Overrides .ufo/settings.yml.
```
