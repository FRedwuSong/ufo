## Usage

    ufo tasks build

## Description

builds task definitions

## Examples

    ufo tasks build

Builds all the task defintiions.

Note all the existing ufo/output generated task defintions are wiped out.


## Options

```
[--pretty], [--no-pretty]  # Pretty format the json for the task definitions
                           # Default: true
```
