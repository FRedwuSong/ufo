## Usage

    ufo releases SERVICE

## Description

Show possible 'releases' or task definitions for the service.

## Example

    $ ufo releases
    Recent task definitions for this service:
      demo-web:287
      demo-web:286
      demo-web:285
      demo-web:284
      demo-web:283
      demo-web:282
      demo-web:281
      demo-web:280
      demo-web:279
      demo-web:278

## Options

```
[--verbose], [--no-verbose]  
[--mute], [--no-mute]        
[--noop], [--no-noop]        
[--cluster=CLUSTER]          # Cluster.  Overrides .ufo/settings.yml.
```
