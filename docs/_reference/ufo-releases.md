## Usage

    ufo releases SERVICE

## Description

Show possible 'releases' or task definitions for the service.

## Example

    $ ufo releases
    Recent task definitions for this service:
      demo-web:86
      demo-web:85
      demo-web:84
      demo-web:83
      demo-web:82
      demo-web:81
      demo-web:80
      demo-web:79
      demo-web:78
      demo-web:77
      demo-web:76
    $


## Options

```
[--verbose], [--no-verbose]  
[--mute], [--no-mute]        
[--noop], [--no-noop]        
[--cluster=CLUSTER]          # Cluster.  Overrides .ufo/settings.yml.
```
