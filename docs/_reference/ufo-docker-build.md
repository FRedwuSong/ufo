## Usage

    ufo docker build

## Description

builds docker image

Examples:

    ufo docker build
    ufo docker build --push # also pushes the image to the docker registry

Docker image tongueroo/hi:ufo-2016-10-21T15-50-57-88071f5 built.


## Options

```
[--push], [--no-push]  
```
