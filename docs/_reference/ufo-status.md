## Usage

    ufo status SERVICE

## Description

Status of ECS service.  Essentially, status of CloudFormation stack


## Options

```
[--verbose], [--no-verbose]  
[--mute], [--no-mute]        
[--noop], [--no-noop]        
[--cluster=CLUSTER]          # Cluster.  Overrides .ufo/settings.yml.
```
